This repository contains the data and code used in the empirical analysis of the paper titled “Increasing the Adoption of Electric Vehicles May Exacerbate Carbon Emissions from Power Plants in China”.

The dataset is a panel data file that includes all variables used in the empirical analysis.

The Stata do-file is used to replicate all empirical results reported in the paper, including the main regression analyses, robustness checks, and the generation of all figures and charts.

To reproduce the results, the analysis can be fully implemented by running the provided Stata do-file using the accompanying dataset.
